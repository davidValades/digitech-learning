Java vs JavaScript

Java JavaScript
Lenguaje compilado (Bytecode ejecutado en la JVM). Lenguaje interpretado (Ejecutado por el motor del navegador).
Tipado estático y fuerte (debes declarar el tipo de dato). Tipado dinámico y débil (el tipo se asigna en tiempo de ejecución).
Orientado a clases. Orientado a objetos basado en prototipos.
