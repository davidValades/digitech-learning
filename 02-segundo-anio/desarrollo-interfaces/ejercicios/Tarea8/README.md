## 📝 Descripción de la Tarea

Este proyecto forma parte de la evaluación práctica del módulo de Desarrollo de Interfaces. El objetivo principal es demostrar la capacidad de integrar frameworks CSS modernos y comprender sus diferencias arquitectónicas y prácticas.

La tarea se divide en dos bloques integrados en un único documento web:
1.  **Implementación Práctica:** Refactorización de un formulario HTML desarrollado previamente en clase, aplicando **Tailwind CSS** mediante CDN para su estilizado *utility-first*.
2.  **Análisis Teórico:** Respuestas a las cuestiones teóricas solicitadas, maquetadas y estilizadas directamente en la interfaz.

---

## 📂 Estructura de Archivos

El proyecto entregado (`dValades.zip`) ha sido optimizado y contiene la siguiente estructura:

*   `index.html`: Archivo principal que incluye:
    *   El formulario interactivo estilizado con Tailwind CSS.
    *   Definición técnica de Tailwind CSS y análisis de ventajas/desventajas.
    *   Cuadro comparativo detallado entre Bootstrap y Tailwind CSS (uso, curva de aprendizaje, tamaño, JavaScript e instalación).
*   `README.md`: Documentación actual del proyecto.

---

## ⚙️ Tecnologías Utilizadas

*   **HTML5:** Estructura semántica del formulario y de la sección teórica.
*   **Tailwind CSS:** Framework CSS *utility-first* cargado vía CDN (`<script src="https://cdn.tailwindcss.com"></script>`).
